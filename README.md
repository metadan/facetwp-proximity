FacetWP - Proximity Facet
=======================

A [FacetWP](https://facetwp.com/) facet add-on for filtering posts by proximity. It uses the [Google Places API](https://developers.google.com/places/documentation/autocomplete) to display an autocomplete search box.

1.2.1

* Added `facetwp_proximity_radius_options` filter for customizing the Radius dropdown choices
